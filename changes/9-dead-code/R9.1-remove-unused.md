# R9.1 — Remove dead sparkles.tsx + unused dependencies

**Status:** planned
**Files:** `src/components/ui/sparkles.tsx` (delete), `package.json` (drop deps), `pnpm-lock.yaml`

## Spec

**Problem:** Several heavy dependencies and one component file ship but are never imported
(verified via grep across `src/`): `sparkles.tsx` (unused; sole `@tsparticles/*` consumer),
`@tsparticles/engine|react|slim`, `three-globe`, `@react-three/drei`, `@tabler/icons-react`.
They bloat install size, the lockfile, and the eslint surface (sparkles.tsx has an `any` error).

**Approach:**
1. `git rm src/components/ui/sparkles.tsx`.
2. `pnpm remove @tsparticles/engine @tsparticles/react @tsparticles/slim three-globe
   @react-three/drei @tabler/icons-react`.
3. `pnpm build` must still succeed (tsc + vite) — this is the safety net proving nothing
   referenced them. If anything breaks, revert.

KEEP (verified used): `three` + `@react-three/fiber` (canvas-reveal-effect), `cobe` (globe),
`simplex-noise` (wavy-background), `react-parallax-tilt` + `aceternity.tsx`.

Decision: remove only verified-zero-import items; rely on the build as proof. Recorded.

State: none.
