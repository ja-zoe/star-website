## Spec

**Problem:** The "Our Projects" cards on the home page predate the set-10 accent system. On hover
the `CanvasRevealEffect` shows `containerClassName="bg-[#9D2626]"` (STAR red) for **all three**
cards, and the per-card `colors` (reveal dots) are stale/old: robotics `[[3,208,16]]`, weather
`[[0,145,255]]`, cubesat `[[0,0,0]]` (black â†’ invisible). So none of the cards reflect their
project's signature accent (amber / emerald / sky), and CubeSat's reveal looks broken. Verified
live: hovering CubeSat reveals a red wash with no visible dots
(scratchpad/card-hover-cubesat.png).

**Approach:** Drive the cards from the single source of truth (`projectConfig`) so the home cards
match their pages.
1. **`projectConfig.ts`** â€” add two fields to `ProjectConfig`:
   - `accentRgb: [number, number, number]` â€” the accent as RGB for `CanvasRevealEffect.colors`
     (cubesat `[245,165,36]`, robotics `[52,211,153]`, weather `[56,189,248]`).
   - `cardRevealBg: string` â€” a Tailwind arbitrary bg class for the reveal backdrop, a deep tint
     of the accent so the bright accent dots pop (not full-bright). Values:
     cubesat `bg-[#2b1d02]`, robotics `bg-[#06291f]`, weather `bg-[#06222e]`. (â‰ˆ accent at ~12%
     over black; tuneable.)
2. **`ProjectsSection.tsx`** â€” stop hardcoding three `<ProjectCard>`s with literal colors. Import
   the three configs and render the cards from them, passing `colors={[cfg.accentRgb]}` and a new
   `revealBg={cfg.cardRevealBg}` prop. Keep the existing icons (rover/satellite/balloon) and
   hrefs. Order stays Robotics, Weather Balloon, Cubesat (current visual order) unless trivially
   keyed off config order â€” match current order.
3. **`ProjectCard.tsx`** â€” add optional `revealBg?: string` prop; use it for the
   `CanvasRevealEffect` `containerClassName` instead of the hardcoded `bg-[#9D2626]` (fallback to
   the current red if not provided, so nothing else breaks). No other change; the lazy three.js
   boundary (set 2 R2.2) stays.

Decision: dots = accent (bright), backdrop = deep accent tint â†’ the hover now reads as the
project's color and visibly sparkles. Cohesion comes from sourcing both from `projectConfig`
(same place the pages theme from). Recorded; no Open Question.
