# R12.1 — STAR red brand thread on project pages

**Status:** planned
**Files:** `src/components/project/ProjectShell.tsx` (all three changes live here — shared kit)

## Spec

**Problem:** After set 10/11, each project page leads with its signature `--accent`
(amber/emerald/sky) but carries no STAR red, so the pages don't visibly tie back to the STAR
brand. We want red `#9D2626` woven through the shared chrome of every project page so the three
read as one branded family, without diluting each page's accent identity (design/tokens.md
`--brand`; design/components.md › STAR red brand thread).

**Approach:** All three changes are in `ProjectShell.tsx` (the shared kit → automatically applies
to all three pages identically). Define `const BRAND = "#9D2626"` in the file.
1. **Hero eyebrow STAR mark.** Import `Star` from lucide. Before the eyebrow `<p>` text, render a
   small filled red star: `<Star className="w-4 h-4 fill-[#9D2626] text-[#9D2626]" aria-hidden />`
   inline with the eyebrow (wrap eyebrow in a flex row: star + text). Eyebrow text stays
   `text-[var(--accent)]`.
2. **Red strand in the hero wave.** Change the `WavyBackground` colors from `[accent, "#0a0a0a"]`
   to `[accent, BRAND, "#0a0a0a"]` so STAR red is blended into every hero's signature wave.
3. **"A STAR Project" brand tie.** Add a slim strip as the last thing in the ProjectShell root
   (after the Status/Join `<section>`, before the closing `</div>`): centered, `border-t
   border-white/10 pt-8 mt-4`, an `<a href="/">` with a small filled red `Star` + muted text
   "A STAR Project · Space Technology Association of Rutgers" (`text-white/45 text-sm`, hover
   `text-white/70`). Identical on all three pages → the brand sign-off. The star is red; text
   muted white so it doesn't fight the accent.

Constraints: red touches ONLY these three shared elements. Accent-owned parts (section labels,
stat numerals, mission rule, subteam wash/modal, CTAs) stay `--accent`. No new per-project config
(red is constant). Keep `prefers-reduced-motion` behavior (WavyBackground already gated). No
photos.

State: none (static markup + a constant).
