# R7.2 — Social share image (1200×630)

**Status:** tests passing
**Files:** `scripts/optimize-images.mjs` (extend), `public/og-image.png` (new, generated),
`index.html` (og:image / twitter:image)

## Spec

**Problem:** `og:image` and `twitter:image` point at `star-image.png` (404×292) — below the
recommended 1200×630 for social cards, so shared links render small / awkwardly cropped on
Discord/LinkedIn/iMessage/etc. For a club that recruits by sharing links, the card matters.

**Approach:** Extend `scripts/optimize-images.mjs` to composite a 1200×630 `public/og-image.png`:
solid black (#000000) canvas, the STAR logo (`star-logo-transparent.png`) centered and scaled
to ~520px wide (sharp: create base via `{create:{width:1200,height:630,channels:4,background:
"#000000"}}`, `composite` the resized logo centered). Export optimized PNG. Then update
`index.html` `og:image` + `twitter:image` → `/og-image.png` (also set `og:image:width` 1200 /
`og:image:height` 630 for scrapers).

Decision: simple on-brand card (logo on black) — matches the site's dark identity; no text
overlay needed (the og:title/description carry the words). Recorded; no Open Question.

State: none.
