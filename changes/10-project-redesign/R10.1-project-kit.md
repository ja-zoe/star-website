## Spec

**Problem:** The three project pages are flat stacked centered text (hero tagline → one
overview paragraph → subteam cards → "meeting times" placeholder). They don't feel curated to
each project or like a cohesive series.

**Approach:** Build a shared editorial "mission dossier" kit (design/components.md › Project page
kit) and render all three pages through it, differentiated only by a per-project config +
`--accent`.

1. **`projectConfig.ts`** — `ProjectConfig` type + 3 configs (cubesat/robotics/weatherBalloon):
   `{ eyebrow, name, tagline, accent (hex), motifIcon (lucide), stats: {value,label}[],
   mission: string[] (paragraphs from existing copy), statusBody, ctaHref, ctaLabel }`.
   - CubeSat: accent `#F5A524`, icon `Satellite`, stats [8 Subteams · UNP-backed · Microgravity
     science · Rutgers' 1st sat], mission = existing SPICEsat paragraphs.
   - Robotics: accent `#34D399`, icon `Bot`, stats [3 Subteams · NASA Lunabotics · Autonomous ·
     Lunar regolith], mission = existing Lunabotics paragraph (typo already fixed).
   - Weather Balloon: accent `#38BDF8`, icon `Wind` (or `CircleDashed`), stats [80,000+ ft · 2
     Subteams · Every semester · Near-space], mission = existing high-altitude paragraph.
2. **`SectionLabel.tsx`** — mono uppercase `text-white/50` with a leading `var(--accent)` tick;
   props `{ index, children }` → "01 / THE MISSION".
3. **`ProjectShell.tsx`** — sets `style={{ ['--accent']: config.accent }}` on root; renders:
   Hero (WavyBackground recolored to `[accent, '#0a0a0a']`, eyebrow + `<h1>` + tagline + accent
   CTA → `#mission`, small motif icon) → StatBand (bordered `border-y border-white/10`, accent
   numerals + uppercase labels, dividers) → Mission (`id="mission"`, SectionLabel + left-aligned
   prose in a centered max-w-3xl column, accent rule) → Subteams (SectionLabel + `children`) →
   Status/Join (SectionLabel + statusBody + meeting note + accent Discord CTA). Keeps `<Seo>` in
   the page. h1 stays the single page h1; section titles are h2 (keep R6.1 outline).
4. **Pages** — each becomes: `<Seo/>` + `<ProjectShell config={…}>{<Subteam/> cards}</ProjectShell>`.
5. **Subteam card theming** — add `.accent-wash` to App.css (radial accent glow via
   `color-mix(in srgb, var(--accent) …)`) and replace the 13 subteam `className:
   "bg-radial from-…/20 to-transparent"` with `className: "accent-wash"`. StarStat + SubteamModal
   logic unchanged (SubteamModal still appends ` h-60`). Home AboutStarSection StarStats keep
   their own red/blue washes — untouched (no `--accent` there → `.accent-wash` not applied).

Decision: reuse existing subteam modal content + StarStat/WavyBackground components (re-themed,
not replaced) — refine, don't reinvent. No photos. Recorded; no Open Question.
